Disciplinas

O que você vai aprender aqui!

Programação web

HTML + CSS + JS + PHP para você dominar os fundamentos da programação WEB

IoT

Você vai aprender na prática a programar microcontroladores capazes de automatizar tarefas, usando lógica de programação e construir pequenos circuitos eletrônicos reais

Lógica de programação e paradigma OO

Você vai testar sua lógica resolvendo problemas reais do dia a dia com programação e aprender a modelar seus problemas em classes e objetos

Banco de dados

Você vai aprender a armazenar dados de suas aplicações, construir memória para elas, e como construir e gerenciar essa memória, em conjunto com um sistema

Interfaces gráficas

Você vai construir intefaces gráficas completas para programas desktop, com várias janelas, muita lógica de programação e muita funcionalidade para o usuário final

Mobile

Você vai criar aplicativos para celulares e smartphones, e instigar sua criatividade na hora de criar telas e funcionalidades com as ferramentas presentes nos celulares atuais, como giroscópios e acelerômetros

Python

Você vai aprender a linguagem do momento, e aprender a usar ela para análise e ciência de dados, criar e treinar inteligências artificiais robustas, trabalhar com aprendizado de máquina, mineiração de dados da web, manipulação de imagens e muitas outras coisas

Engenharia de sotfware

Você vai aprender sobre gestão e gerência de software, e ficar pronto para trabalhar na área de gestão de sistemas de uma empresa, onde você ficará responsável por gerenciar times de desenvolvimento, trabalhar com planejamentos e cronogramas, tomar decisões estratégicas e importantes sobre o software e a sua equipe

Sobre o Curso

Saiba o porquê fazer TDS

O curso de Tecnologia em Desenvolvimento de Sistemas tem por objetivo a formação de profissionais para atuação em planejamento, análise, utilização e avaliação de modernas tecnologias de informação aplicadas às áreas administrativas e industriais, em organizações públicas e privadas.

Com ele você sairá um egresso pronto para atuar profissionalmente no mercado, alinhado com as exigências das empresas e a par das evoluções contínuas da área.
